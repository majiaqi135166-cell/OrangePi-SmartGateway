# 🚀 Orange Pi Smart Gateway

基于 Orange Pi Zero 3 的智能网关原型开发，专注于嵌入式 Linux 底层交互与性能监控。

## ✨ 当前功能
- **System Monitor**: 实时监控 CPU 温度、负载。
- **Hardware Control**: 基础 GPIO 驱动与 LED 状态控制。

## 🔧 开发环境
- **硬件**: Orange Pi Zero 3 (Allwinner H618)
- **系统**: Armbian / Ubuntu
- **语言**: C Language
- **部署**: Git-based hardware-to-cloud workflow
